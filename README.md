# LaundryPro Login Page

## 📌 Project Description
This project is a modern and responsive Login Page for the LaundryPro web application.  
It provides a clean user interface with essential authentication features.

## 🚀 Features
- Full screen layout using 100vh
- Left side image section
- Right side login form
- Email and Password input fields
- "Forgot Password?" link
- "Not a member? Register Now" option
- Styled login button with gradient
- Clean and user-friendly UI

## ▶️ How to Run the Project
1. Download or clone this repository
2. Open the project folder
3. Make sure all files are in the same directory
4. Double click on `index.html`
5. The login page will open in your browser

## ⚠️ Important Notes
- Ensure the image file (`laundry.jpg`) is present in the folder
- CSS file must be properly linked to HTML
- This is a frontend-only project (no backend functionality)

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox)
